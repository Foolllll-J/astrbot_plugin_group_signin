# ✅ 群打卡插件

![License](https://img.shields.io/badge/license-GPL--3.0-blue)
![Python](https://img.shields.io/badge/python-3.10+-blue.svg)
![AstrBot](https://img.shields.io/badge/framework-AstrBot-orange)

一款为 [AstrBot](https://github.com/AstrBotDevs/AstrBot) 设计的QQ群打卡插件，支持定时打卡、一键打卡、智能通知和详细的统计功能。

---

## ✨ 功能特性

* **🎯 简化配置**: 使用简单的时分格式设置定时任务，每日固定执行
* **⚡ 一键打卡**: 对所有配置群组一次性打卡，自动间隔防风控
* **🔔 智能通知**: 支持成功通知和每日报告两种模式，异常强制通知
* **📊 详细统计**: 记录总计和每日打卡数据

---

## 📖 使用指南

### 🎯 管理员打卡指令

**立即打卡**:

```
/立即打卡
```

在群聊中使用，立即为当前群打卡。

```
/立即打卡 123456789
```

为指定群号打卡。

**一键打卡**:

```
/一键打卡
```

对所有设置了定时任务的群组执行一次打卡，自动间隔防风控。

---

### ⏰ 定时打卡管理 (仅管理员)

**添加定时任务**:

```
/添加定时打卡 <时间HH:MM> <群号>
```

**示例**:

```
/添加定时打卡 09:00 123456789
```

每天9点为群 123456789 打卡。

**查看定时任务**:

```
/查看定时打卡
```

显示所有已创建的定时任务。

**删除定时任务**:

```
/删除定时打卡 <群号>
```

删除指定群号的定时任务。

---

### 📊 统计信息

**查看统计**:

```
/打卡统计
```

显示打卡统计数据，包括：
- 总计统计（所有打卡记录）
- 今日统计（当日打卡情况）
- 配置信息（定时任务数、通知设置）

---

### ❓ 帮助

```
/打卡帮助
```

显示详细的使用说明。

## 🔧 时间格式说明

所有时间配置使用简单的 `HH:MM` 格式：

- **HH**: 小时 (00-23)
- **MM**: 分钟 (00-59)

**示例**:

| 时间 | 说明 |
| :--- | :--- |
| `09:00` | 每天早上9点 |
| `18:30` | 每天下午6点30分 |
| `00:00` | 每天零点 |
| `23:59` | 每天23:59 |
